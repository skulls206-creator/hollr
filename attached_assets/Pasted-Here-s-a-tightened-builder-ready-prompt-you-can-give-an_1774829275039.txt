Upgrade my HOLLR CHAT app by adding 4 Discord-inspired features, but make them feel native to HOLLR, not like a clone.

Feature 1: Voice connection card
When a user is connected to a voice channel, show a compact “Voice Connection” popover/card with: current ping in ms, average ping in ms, endpoint/server label, device name, and a small live line chart showing recent latency trend from good to bad.
​
The chart should visually indicate connection quality using thresholds, for example green for healthy latency, yellow for unstable, red for poor.
​
The card should be accessible from the active voice status area and update in near real time.

Feature 2: Expanded voice/channel stats modal or popover
Add a second detailed “Connection Stats” panel for the active voice session showing duration, participants count, audio send bitrate, audio receive bitrate, video send bitrate, video receive bitrate, packet loss, latency (RTT), and jitter.
​
Update stats every second or on a reasonable polling interval and make sure empty values degrade gracefully when video or receive streams are not active.
​

Feature 3: Floating screenshare preview in chat
When someone in the active voice channel is screensharing, show a small floating mini-player/preview anchored in the bottom-right of the chat window.
​
The preview should not block message composition, should be draggable or dismissible, and should open the full stream view when clicked.
​
Add states for: live screenshare, paused/ended share, hidden/minimized preview, and no active screenshare.

Feature 4: Per-channel age restriction toggle
In each server’s text-channel settings, add an “Age-restricted (18+) / NSFW” toggle for individual hashtag channels such as #general.
​
This should be configurable per channel, not just per server.
​
When enabled, underage users must be blocked and adults should see a warning/confirm screen before entering the channel.
